# k5-Fullpageslide
超精简全屏翻页h5框架 Fullpageslide

>[example](//nervouself.github.io/k5-fullpageslide/)

## usage

```html
    <script src="${yourpath}/Fullpageslide.js"></script>
    <script>
        new Fullpageslide('wrap',options)
    </script>
    /* 
     * options {
     *   (最小滑动间距) deltY: 60,   
     *   (翻页动画时间) duration: '.5s',
     *   (翻页动画曲线) timing: 'cubic-bezier(0.7, 0, 0.3, 1)'
     * }
     *
     */ 
```
